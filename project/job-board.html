Overview

Marketplace where companies post jobs and candidates apply. Features advanced full-text job search, application state machine, and Gemini-powered resume parsing.

How it Works

When a PDF is uploaded, Gemini extracts structured JSON data (skills, experience). A PostgreSQL full-text search index allows recruiters to search millions of resumes efficiently, while Spring Batch runs nightly to match new candidates against open roles.

Who is it for?

HR agencies, recruiters, and corporate talent acquisition teams aiming to automate the initial screening and matching phase of hiring.

Architecture & Stack

  • Backend: Java 17, Spring Boot 3.3, Spring Batch
  • Frontend: React 18, React Query
  • AI & DB: Google Gemini API, PostgreSQL (Full-Text Search)
